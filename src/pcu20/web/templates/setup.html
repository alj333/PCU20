Sinumerik PCU20 (810D / 840D) Setup

The PCU20 connects to this server as a client. You need to configure the CNC's network settings to point to this machine.

1

Configure CNC Network Settings

On the Sinumerik HMI panel:

  1. Press Menu Select until you see the IBN (Commissioning) softkey
  2. Press IBN, then navigate to CNC ISO
  3. Select Network connections
  4. Set the network configuration:
    IP AddressAssign a static IP on your LAN (e.g. 192.168.1.100)
    Subnet Mask255.255.255.0
    GatewayYour network gateway (e.g. 192.168.1.1)
2

Configure Logical Drive

Map a network drive to this server:

  1. Navigate to Logical drives in the network configuration
  2. Add a new drive mapping:
    Drive letterChoose an available letter (e.g. E:)
    Server IP{% for ip in server_ips %}{{ ip }}{% if not loop.last %} or {% endif %}{% endfor %}
    Port{{ pcu20_config.base_port }}
    UsernamePCU20_USER
    Passwordpcu20 (default)
3

Configure Softkeys (Optional)

Set up function key labels on the HMI to quickly access the network drive:

  1. Navigate to Softkey labels
  2. Assign Row 1 / Row 2 labels for the mapped drive
4

Restart HMI & Verify

Restart the HMI. The PCU20 should connect to this server automatically. Check the Dashboard for a new connection.

If it doesn't connect, verify the network cable, IP settings, and that this server is running.

Fanuc CNC Setup (30i / 16i / 0i-MD)

This server connects to the Fanuc CNC via FOCAS2. You need to configure the CNC's Ethernet port and add it to your pcu20.toml.

1

Configure CNC Ethernet

On the Fanuc CNC operator panel:

  1. Press SYSTEM, then select the ETHPRM (Ethernet Parameters) screen
  2. Navigate to COMMON tab and set:
    IP ADDRESSAssign a static IP (e.g. 192.168.1.50)
    SUBNET MASK255.255.255.0
    ROUTER IPYour gateway (e.g. 192.168.1.1)
2

Enable FOCAS2 Server

Still in Ethernet Parameters:

  1. Navigate to the FOCAS2 or PORT tab
  2. Set:
    PORT NUMBER8193 (default FOCAS2 port)
    TIME INTERVAL10 (seconds, keepalive)
  3. Cycle power on the CNC to apply changes
3

Set CNC Parameters

Some parameters must be set for FOCAS2 to work:

ParameterBitValueDescription
0020Bit 0 (NE9)0Enable embedded Ethernet
0020Bit 6 (NE6)0Enable FOCAS2/Ethernet function

Parameters vary by controller model. Consult your Fanuc Parameter Manual.

4

Add Machine

Use the Configured Machines form at the top of this page to add the Fanuc CNC. Fill in the IP address, select the CNC type, and click Add Machine. The config is saved automatically.

Restart this server after adding the machine so FOCAS2 polling connects.

5

Verify Connection

Use the Connection Test tool above to verify the CNC is reachable on port 8193. Then restart this server — the machine should appear on the Dashboard.

DMG Mori MAPPS Setup

Mori MAPPS machines use a Fanuc 31i controller underneath. Connection is via FOCAS2, same as Fanuc — but Ethernet access is through the MAPPS interface.

1

Locate the FOCAS2 Ethernet Port

On Mori machines, there are typically two Ethernet ports:

  • HMI port — on the MAPPS PC (for MTConnect / Windows networking)
  • FOCAS2 port — on the Fanuc NCU (metal bracket, usually on the back/side of the control cabinet)

Connect your network cable to the FOCAS2 port on the NCU, not the HMI port.

2

Configure Ethernet via MAPPS

  1. On the MAPPS screen, navigate to Setup → Communication
  2. Select the Ethernet or FOCAS tab
  3. Set:
    IP AddressStatic IP (e.g. 192.168.1.53)
    Subnet Mask255.255.255.0
    Port8193
  4. If not accessible via MAPPS, configure through the underlying Fanuc panel (same as Fanuc guide above — press SYSTEM → ETHPRM)
3

Add Machine

Use the Configured Machines form at the top of this page. Set the CNC type to Mori MAPPS.

4

Verify & Restart

Test the connection with the tool above, then restart this server. The MAPPS machine will appear with a FOCAS2 badge on the dashboard.

Note: Some older Mori machines may require the MAPPS HMI to be fully booted before FOCAS2 becomes available. Allow 2-3 minutes after machine power-on.
